# Task 1 — Calculator Application

> **SaiKet Systems | Java Internship Program**

---

## 📋 Description

A console-based calculator that supports six arithmetic operations with robust
input validation and division-by-zero exception handling.

## ▶️ How to Compile & Run

```bash

cd Task1_Calculator

# 2. Compile
javac Calculator.java

# 3. Run
java Calculator
```

**Requirements:** Java 8 or higher (`java -version` to check).

## ⚙️ Edge Cases Handled

- **Division / Modulus by zero** → Custom `ArithmeticException` message
- **Non-numeric input** → `NumberFormatException` caught, re-prompts user
- **Invalid menu choice** → Graceful error message, loop continues
- **Floating-point numbers** → Both integer and decimal inputs supported

## 📐 Method Reference

| Method | Signature | Description |
|--------|-----------|-------------|
| `add` | `(double a, double b) → double` | Returns a + b |
| `subtract` | `(double a, double b) → double` | Returns a − b |
| `multiply` | `(double a, double b) → double` | Returns a × b |
| `divide` | `(double a, double b) → double` | Returns a ÷ b; throws on b=0 |
| `modulus` | `(double a, double b) → double` | Returns a % b; throws on b=0 |
| `power` | `(double base, double exp) → double` | Returns base^exp via Math.pow |
